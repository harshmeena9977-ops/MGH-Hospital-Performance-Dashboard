# Hospital Dashboard Requirements

## 🛠️ Tools and Libraries

### Database Connection
- mysql-connector-python 8.0.33
- MySQL Server 8.0+

### Data Analysis (Optional)
- pandas 1.5.0+
- numpy 1.24.0+
- matplotlib 3.7.0+
- seaborn 0.12.0+

### Dashboard Tool
- Power BI Desktop (Latest version)
- Windows 10/11 for optimal performance

## 📊 Data Requirements

### Database Schema
- Database: hospital_db
- Tables: encounters, patients, procedures, payers, organizations
- Primary Keys: Id fields in each table
- Relationships: Proper foreign key connections

### Data Volume
- Encounters: ~28,000 records
- Patients: ~1,000 records  
- Procedures: ~48,000 records
- Date Range: 2011-2022 (11 years)
